UZYHUB
GUIDE
Guide d'installation
Bienvenue sur le guide d'installation de l'application mobile du Panel Admin du Uzy Hub.
ÉTAPE 1 Autoriser les sources inconnues
Vérifier que votre application de fichier (My Files, Google Files, etc...) à l'autorisation d'installer des applications de source inconnue.
ÉTAPE 2 Lire le guide
Lisez attentivement le reste du guide !
ÉTAPE 3 Télécharger l'application
Lorsque vous installerez l'application cliquez sur "Télécharger quand même".
ÉTAPE 4 Lancer l'installeur
Lorsque vous appuyez sur "Télécharger quand même" l'installeur se lancera et demandera "Voulez-vous installer cette application ?" Appuyez sur "Installer".
ÉTAPE 5 Ouvrir l'application
Une fois l'installation terminée appuyez sur "Ouvrir".
ÉTAPE 6 Application ouverte
Après avoir appuyé sur "Ouvrir" l'application s'ouvrira.
ÉTAPE 7 Installation confirmée
Allez sur la page application et vous verrez que l'application est officiellement installée sur votre téléphone.
